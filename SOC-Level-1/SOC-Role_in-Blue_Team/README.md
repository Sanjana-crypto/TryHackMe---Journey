## Key Concepts

### L1 Analyst
- Monitors alerts
- Performs initial investigation
- Escalates incidents

### L2 Analyst
- Conducts deeper analysis
- Validates threats
- Handles incident investigations

### L3 Analyst
- Advanced threat hunting
- Malware analysis
- Security engineering

### Incident Response
- Identification
- Containment
- Eradication
- Recovery
- Lessons Learned
