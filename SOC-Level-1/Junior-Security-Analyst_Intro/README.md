## Key Concepts

### Security Operations Center (SOC)

A SOC is a centralized team responsible for monitoring, detecting, analyzing, and responding to cybersecurity incidents.

### Alert Triage

The process of reviewing security alerts and determining their severity and legitimacy.

### Incident Response

A structured approach to identifying, investigating, containing, and recovering from security incidents.

### Threat Intelligence

Information about attackers, their tactics, techniques, and procedures (TTPs) used to improve security defenses.
